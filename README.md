# QuickBooks API Integration

## Overview
This project is a Node.js application that connects to the QuickBooks API to manage business financial data. It demonstrates back-end development skills, API integration, and secure authentication using OAuth2.

## Features
- Connects to QuickBooks API using **OAuth2** for secure authentication
- Stores and retrieves financial data in a SQL database
- Provides API endpoints for querying invoices, customers, and payments
- Includes basic tests to validate functionality

## Tech Stack
- **Node.js** and **Express.js**
- **SQL** (SQLite/MySQL/PostgreSQL, specify which you used)
- **QuickBooks API** with OAuth2 authentication
- **JavaScript**
